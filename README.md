# Дипломная работа Автоматизация теста к API
    Для запуска теста должны быть установлены пакеты pytest и requests
    Запуск теста выполняется командой pytest
Шаги автотеста:
Выполнить запрос на создание заказа.
Сохранить номер трека заказа.
Выполнить запрос на получение заказа по треку заказа.
Проверить, что код ответа равен 200.

SQL-запрос
    Проверка отображениения заказа, созданного в базе данных. 
    Подтверждение, что статусы заказов корректно записываются в базу данных
